# 🎲 Ludo Game for Telegram Mini App

Полноценная онлайн-игра Ludo ("Не гони меня") для Telegram Mini App.

## 📦 Структура проекта

```
ludo-game/
├── package.json      # Зависимости
├── server.js         # Сервер (Node.js + Socket.io)
└── public/
    └── index.html    # Клиент (HTML5 Canvas)
```

## 🚀 Быстрый запуск

### 1. Установка зависимостей

```bash
cd ludo-game
npm install
```

### 2. Запуск сервера

```bash
npm start
```

Сервер запустится на порту 3000 (или PORT из env).

### 3. Открытие в браузере

```
http://localhost:3000
```

## 📱 Подключение к Telegram Mini App

### Шаг 1: Создай бота в @BotFather

1. Напиши @BotFather
2. `/newbot` — создай бота
3. Получи токен

### Шаг 2: Настрой Mini App

```
/newapp
```

- Выбери своего бота
- Введи название
- Введи описание
- **Важно:** в поле URL укажи адрес твоего сервера (например: `https://your-domain.com`)

### Шаг 3: Запусти на сервере

Залей файлы на любой VPS/хостинг с Node.js:
- [Railway](https://railway.app) (бесплатно)
- [Render](https://render.com) (бесплатно)
- [Heroku](https://heroku.com)
- Любой VPS (DigitalOcean, Hetzner, etc.)

**Важно:** нужен HTTPS для Telegram Mini App!

### Шаг 4: Подключи бота к чату

Добавь бота в групповой чат. Игроки открывают Mini App через меню бота.

## 🎮 Как играть

1. **Создать игру** — один игрок создаёт комнату, получает код
2. **Присоединиться** — остальные вводят код комнаты (4-5 символов)
3. **Готов** — все нажимают "ГОТОВ"
4. **Старт** — хост нажимает "СТАРТ"
5. **Игра** — по очереди бросай кубик (ROLL) и кликай по фишке чтобы походить

### Правила:
- Чтобы выйти из базы — нужно выбросить **6**
- Если выпала **6** — ходишь ещё раз
- На **звёздочках** нельзя убить
- На **цветной дорожке домой** нельзя убить
- Если встал на клетку с чужой фишкой — она умирает и возвращается в базу
- Кто первый провёл все 4 фишки до центра — победил!
